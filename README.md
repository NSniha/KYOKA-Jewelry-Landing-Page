# Kyoka — Vintage Designer Jewellery Landing Page

A premium, elegant, and fully responsive jewellery landing page built for **Kyoka**, a luxury vintage designer jewellery brand.  
This project focuses on refined visual storytelling, immersive product presentation, and modern front-end interactions using **HTML**, **Tailwind CSS**, **custom CSS**, and **vanilla JavaScript**.

The design combines a soft luxury aesthetic with interactive shopping-focused sections such as a **hero slider**, **discover categories slider**, **trending jewellery carousel**, **cart drawer**, **Instagram-inspired gallery**, and a polished footer experience.

---

## Preview

Kyoka is designed to create a high-end editorial shopping experience with:

- A cinematic **hero banner slider**
- Curated **jewellery category showcase**
- Interactive **trending product slider**
- Variant switching for **gold / silver**
- Functional **shopping cart drawer**
- Elegant **marquee branding section**
- Premium **Instagram-inspired gallery**
- Clean and sophisticated **responsive footer**
